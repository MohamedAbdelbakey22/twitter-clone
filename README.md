# 🚀 X (Twitter) Clone - Responsive UI

## 🌟 Key Features

* **Pixel-Perfect UI:** A complete recreation of the X interface, including the navigation sidebar, main feed, and trending section.
* **Fully Responsive:** Optimized for all screen sizes (Mobile, Tablet, Desktop) using Tailwind's responsive utilities.
* **Dark Mode Support:** Includes a "Customize Your View" feature to toggle themes and background colors.
* **Interactive Components:** Functional layouts for tweeting, searching, and navigating through the platform.
* **Asset Management:** Organized handling of media and icons for a seamless user experience.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for better SEO and accessibility.
* **Tailwind CSS:** Utility-first framework for rapid and modern styling.
* **JavaScript (ES6+):** For UI interactions and dynamic components.
* **Node.js & npm:** For package management and build automation.

---

## 📁 Project Structure

As shown in the repository, the project follows a clean directory layout:
* `/src`: Source files and main logic.
* `/assets`: Images, icons, and media files.
* `/css`: Custom and compiled stylesheets.
* `tailwind.config.js`: Custom theme and color configurations.
