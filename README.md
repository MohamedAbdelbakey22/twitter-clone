# 🚀 X (Twitter) Clone - Static Responsive UI

A high-fidelity, **static** recreation of the X (Twitter) interface. This project focuses on mastering layout architecture using **Tailwind CSS** and pure **HTML**, demonstrating a "Design-First" approach to web development.

---

## 🌟 Key Features

* **Pixel-Perfect UI:** A precise visual replica of the X interface, including the navigation sidebar, main feed, and the "What's happening" section.
* **100% Responsive Design:** Built from the ground up to be fully responsive across all devices (Mobile, Tablet, and Desktop) using Tailwind's responsive utilities.
* **Modern Layouts:** Uses Flexbox and CSS Grid to manage complex layouts and sidebars seamlessly.
* **Custom Styling:** Customized Tailwind configurations to match X's unique color palette and dark mode aesthetics.
* **Clean Codebase:** Organized folder structure with dedicated directories for assets and source files.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic markup for a clean and accessible structure.
* **Tailwind CSS:** Utility-first framework for rapid, modern, and consistent styling.
* **CSS3:** Custom styles for fine-tuning the user interface.
* **Node.js & npm:** Used for managing Tailwind dependencies and build processes.

---

## 📁 Project Structure

* `/src`: Main source files.
* `/assets`: Images and media used in the feed and profiles.
* `/css`: Compiled Tailwind styles.
* `index.html`: The main entry point of the application.
* `tailwind.config.js`: Custom theme and color configurations.

---

## ⚙️ How to Run

1.  Clone the repository:
    ```bash
    git clone [https://github.com/YourUsername/Twitter-Clone.git](https://github.com/YourUsername/Twitter-Clone.git)
    ```
2.  Install Tailwind dependencies:
    ```bash
    npm install
    ```
3.  Open `index.html` directly in your browser or use a "Live Server" extension.
